### 1. ::作用域运算符(在ros系统开发中经常用到)

相关:生命周期，全局变量，局部变量，命名空间，static，类

tips:

命名空间只能全局命名；命名空间是开放的；命名空间内的函数可以访问空间内的变量，这是全局变量；using namespace xxx;

```cpp
namespace LongName{
}
// 取别名，实际上是复制给ShortName
int main(){
  namespace ShortName = LongName;
}
```
```cpp
namespace A{
  int a;
}
namespace A{
  void func(){}
}
```
```cpp
#include <iostream>
int a =10;
int main()
{
  std::cout<<a<<endl;
  std::cout<<::a<<endl;
}
```
